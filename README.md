# Car Shop

Este repositório contém o projeto "_Car Shop_", requisito em atendimento ao módulo de Back-end, da Trybe. O projeto consiste em aplicar os princípios de Programação Orientada a Objetos (POO) para a construção de uma API com CRUD para gerenciar uma concessionária de veículos.

### Habilidades:
 - colocar em prática os conhecimentos aprendidos sobre POO e MongoDB
